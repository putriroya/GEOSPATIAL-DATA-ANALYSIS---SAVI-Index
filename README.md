# GEOSPATIAL-DATA-ANALYSIS---SAVI-Index
Processing Sentinel-2 Optical Data for Spectral Analysis and SAVI Vegetation Index

This report presents the preprocessing of Sentinel-2 optical imagery and
vegetation analysis for the municipality of Marcilla, located in Navarre, Spain. The
workflow combines Google Earth Engine for image selection, spectral signature analysis,
and vegetation index computation (SAVI), with Python for visualization and time-series
analysis of agricultural plots.
